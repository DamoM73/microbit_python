# Micro:bit Python Tutorials

This repository contains tutorials for teaching Python programming with the BBC Micro:bit. It covers a range of topics, from basic Micro:bit programming to advanced features like using sensors, radio communication, and PiicoDev modules. Each lesson introduces a new concept through hands-on coding exercises.

## Features

- **Comprehensive tutorials** on using Micro:bit in Python.
- Lessons include **display, buttons, sensors**, and external module integration.
- **Solutions and debugging** guides for exercises.
