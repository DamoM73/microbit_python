Exercise Solutions¶

Below you will find a solution to all the exercises from this course. Like all programming, there are countless ways to solve a problem, the solutions presented here are just one way. If you get the outcomes you required, then your code solves the problem.

First Program Exercise 3¶

Without the while statement the micro:bit will display the message and then stop on the Heart. This happens because the code is just sequential. It reaches the end and stops. The while statement is needed to make the micro:bit loop back to the start of the message.

First Program Exercise 4¶

When the micro:bit is unplugged it turns off because it is using the power from the computer. When it is plugged back in, it does nothing, because the code hasn’t been sent from the computer.

Custom Exercise 1¶

The LEDs flash all over the place.

This is caused by the micro:bit processor working faster than the LEDs. By the time an LED has been light up to the correct level, the processor has moved both the x value and the y value far beyond where they need to be. The sleep(10) makes the processor wait until the LED is lit up before continuing in the loop.

Custom Exercise 2¶

 1# display.custom exercise 2
 2

 4
 5display.clear()
 6
 7while True:
 8    for y in range(5):
 9        for x in range(5):
10            for level in range(0,10):
11                display.set_pixel(x,y,level)
12                sleep(10)
13            for level in range(9,-1,-1):
14                display.set_pixel(x,y,level)
15                sleep(10)

Is Pressed Exercise 1¶

 1# button.is_presses exercise 1
 2
 3from microbit import *
 4import time
 5import random
 6
 7# select button
 8button = random.choice(["A","B"])
 9
10# display the countdown
11for num in ["3","2","1"]:
12    display.show(num)
13    sleep(1000)
14
15display.show(button)
16
17# start the timer
18start_time = time.ticks_ms()
19
20# wait for correct button to be pressed
21while True:
22    if button == "A" and button_a.is_pressed():
23        break
24    if button == "B" and button_b.is_pressed():
25        break
26    
27# stop the timer
28end_time = time.ticks_ms()
29
30# calculate the elapsed time in seconds
31elapsed_time = (end_time - start_time) / 1000.0
32
33# display the elapsed time
34display.scroll(str(elapsed_time) + " seconds")

Touch Exercise 1¶

 1# touch exercise 1
 2
 3from microbit import *
 4
 5x = 2
 6
 7pin0.set_touch_mode(pin0.CAPACITIVE)
 8pin2.set_touch_mode(pin2.CAPACITIVE)
 9
10while True:
11    display.set_pixel(x, 2, 9)
12    if pin0.is_touched() and x > 0:
13        x -= 1
14    elif pin2.is_touched() and x < 4:
15        x += 1
16    sleep(100)
17    display.clear()
